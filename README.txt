source.cpp为源文件，语言为C++11

6round.exe为6轮DES，16round.exe为标准DES

共有四种模式，模式1为文件CBC加密，模式2为文件CBC解密，模式3为对单block加密，模式4为对单block解密

模式1和2要输入：输入文件，秘钥，初始向量，输出文件

模式3和4要输入：输入block，秘钥

对于秘钥中的奇偶验证位，不予验证，直接认为其无效

在标准DES下:

明文0x1111111111111111，秘钥0x2222222222222222，加密结果为0x08024fcf811da672，共计27个1

明文0x1111111111111113，秘钥0x2222222222222222，加密结果为0xb9f9ed5026211e4a，共计31个1

明文0x1111111111111111，秘钥0x2222222222222221，加密结果为0xe3243df1b43f3cfa，共计37个1

可以发现，原文或秘钥发生1位有效位的变化时，加密结果几乎每一位都发生了变化，有扩散效应。同时，加密结果中0和1的个数没有明显的偏向性，可以认为是01均衡的
