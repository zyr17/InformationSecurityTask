source.cpp为源文件，语言为C++11

共有三种模式，模式1为RSA+DES加密，模式2为RSA+DES解密，模式3为生成RSA公钥和私钥。

模式1需要输入输入文件，输出文件，公钥n和e，模式2需要输入输入文件，输出文件，私钥n和d。模式3会给出一组随机的公钥和私钥。

源代码共有两部分，DES与RSA。其中DES即为上次作业使用的DES，RSA包含一个高精度类bigbin，重载了 + - * / % < 等运算符。进行加密时，首先会随机生成DES的密钥和初始向量，并使用指定的RSA公钥加密DES的密钥。将加密后的DES密钥和DES初始向量置于加密文件的开头。解密时，从文件头读取加密后的DES密钥和初始向量并解密，然后用DES解密剩余数据。

可执行文件和样例在testdata文件夹中。其中README.txt进行简单说明。
